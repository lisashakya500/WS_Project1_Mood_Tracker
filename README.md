Mood Journal Web App
This project is a Mood Journal Web App. It helps users record how they feel every day and what they did. The user selects a mood using emojis and writes activities. The app saves everything in the browser.

Key Features:
Save Data: Uses localStorage to keep data even after refresh
Add & Delete Entries: User can add and remove mood entries
Search & Filter: User can search activities and filter by mood
Stats Dashboard: Shows total entries, most common mood, and mood distribution
Modern UI: Includes emoji buttons, colors, sliding tabs, and dark/light mode

Local Setup (Simple Steps)
Windows:
Create a folder (example: mood-app)
Create 3 files inside:
index.html
style.css
script.js
Copy your code into these files
Save all files
Double-click index.html to open in browser

MacOS:
Create a folder
Create same 3 files:
index.html
style.css
script.js
Paste the code
Save
Open index.html in browser

Reflection
In this project, I learned how to build a complete web application using HTML, CSS, and JavaScript. Before this, I only knew basic coding, but now I understand how to connect everything together.
One important thing I learned is how to use localStorage. This helped me save user data so it does not disappear when the page refreshes. I also learned how to use JavaScript to update the page automatically without reloading.
Another important part was designing the user interface. I used emojis, colors, and layout to make the app simple and easy to use. Features like search and filter helped me understand how to work with user input.
I also faced some problems, especially when combining features like filtering and searching together. Fixing these problems helped me improve my logic and debugging skills.
Overall, this project helped me learn practical web development skills and gave me confidence to build more projects.
